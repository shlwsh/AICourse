# 应用图标目录

此目录用于存放应用图标文件。

## 所需图标文件

在开发和构建应用时，需要提供以下图标文件：

- `32x32.png` - 32x32 像素的 PNG 图标
- `128x128.png` - 128x128 像素的 PNG 图标
- `128x128@2x.png` - 256x256 像素的 PNG 图标（高分辨率）
- `icon.icns` - macOS 应用图标
- `icon.ico` - Windows 应用图标
- `icon.png` - 通用 PNG 图标（建议 512x512 或更大）

## 临时解决方案

在图标文件准备好之前，可以使用 Tauri CLI 生成默认图标：

```bash
# 从一个源图标生成所有需要的图标格式
tauri icon path/to/source-icon.png
```

或者暂时在 `tauri.conf.json` 中将 `icon` 字段设置为空数组 `[]`。

## 注意事项

- 图标应该是正方形的
- 建议使用透明背景的 PNG 格式
- 源图标分辨率建议至少 1024x1024 像素
